# Light Flyer Cli tool

Command line inteface to create local test network

## Usage 
```
npm install 

npm link
```

## Commands

```
# Help & Commands
lfc -h

#V ersion
lfc -V

# Start default test network
lfc start

# Create test network from cli
lfc set cli
```
